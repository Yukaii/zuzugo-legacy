# Zuzugo 租租狗

租租狗是幫助你租房的最佳服務，它可以透過：

- 捷運站距離時間(**獨家**)
- 通勤時間(**獨家**)
- 各種生活機能方便程度(**獨家**)
- 各種已知的租屋篩選條件，坪數價格房間數開伙...

等條件，自動幫你篩選出適合的物件，並透過 Email、各種聊天服務主動通知你！讓你以最快的速度簽下理想的房屋
